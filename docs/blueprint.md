# **App Name**: FixFast Technician

## Core Features:

- OTP Login: Mobile number based login with OTP verification and session persistence.
- Job Request Management: Accept or reject new job requests with clear display of job details.
- Job Status Updates: Update job status with single tap buttons for quick updates (Reached Location, Inspection Done, Repair In Progress, Repair Completed).
- Customer Navigation: One-tap navigation to customer location using Google Maps integration.
- Job Details: Comprehensive job details including customer information, device details, and problem description.
- Profile Management: View and manage technician profile details (name, phone, service areas).
- Estimated Completion Time: An AI tool which provides estimated time to completion for each job by evaluating equipment type, customer issue, location, and available spare parts inventory.

## Style Guidelines:

- Primary color: Soft blue (#A0BFE0) to evoke trust and efficiency, in line with a professional service.
- Background color: Very light blue (#F0F4F8), creating a clean, uncluttered interface on which the primary will stand out.
- Accent color: Muted purple (#B2A0E0) to draw attention to secondary interactive elements.
- Body and headline font: 'PT Sans' for clear readability and a modern, accessible feel. Note: currently only Google Fonts are supported.
- Use clear, outlined icons for job types, status, and navigation. Icons should be large and easily recognizable.
- Card-based layout for jobs and details, optimized for mobile screens. Use rounded corners and subtle shadows for depth.
- Subtle transitions between screens and states. Avoid heavy animations to ensure smooth performance on low-end devices.
